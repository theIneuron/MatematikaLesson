# Герои уроков началки — комплект для разработчика

Команда: **Мадина** (ведущая) + **Bit** (робот-помощник). Герои — это прозрачные PNG поверх слайда, переключаются по состоянию из логики.

---

## 1. Файлы в комплекте

**Картинки (прозрачный фон, класть в `/public/assets/` или папку ассетов проекта):**
```
madina_pointing.png    — показывает на предметы (задание)
madina_happy.png       — радость (верный ответ)
madina_encourage.png   — подбадривает (ошибка)
madina_celebrate.png   — финал «Молодец!»
bit_hint.png           — подсказывает (экран ошибки)
bit_happy.png          — радость (про запас)
```

**Код:** `LessonTemplate.jsx` — готовый шаблон урока с зашитой логикой.

---

## 2. Подключение (2 минуты)

В `LessonTemplate.jsx` вверху впиши пути к картинкам:
```jsx
const HERO = {
  madina_pointing:  "/assets/madina_pointing.png",
  madina_happy:     "/assets/madina_happy.png",
  madina_encourage: "/assets/madina_encourage.png",
  bit_hint:         "/assets/bit_hint.png",
};
```
Заглушки-рамки заменятся на героев автоматически.

Контент урока меняется в блоке `LESSON` (вопрос, эмодзи, количество, варианты, правильный ответ, правило, подсказка). Логику трогать не нужно.

---

## 3. Правила появления (зашиты в логику)

| Ситуация | Кто на экране |
|---|---|
| Задание / счёт | Мадина (показывает) |
| **Верный ответ** | **только Мадина** (радуется) + правило |
| **Неверный ответ** | Мадина (подбадривает) + **Bit с подсказкой** |
| Финал «Молодец!» | Мадина (празднует) |

**Жёсткие правила:**
- Один герой на экране (исключение — экран ошибки: Мадина + Bit вместе).
- **Bit появляется ТОЛЬКО на неправильном ответе.** На верном — никогда.
- Никакого красного креста и резких звуков на ошибке. Тон мягкий.
- На ошибке кнопки ответа снова активны — даём попробовать сразу.

---

## 4. Размещение на слайде

- Герой — слева или в углу, высота ~25–30% экрана.
- НИКОГДА не перекрывает предметы для счёта, кнопки ответа и текст.
- Одинаковый размер и место на всех слайдах — герой не «прыгает».
- Анимация по желанию (CSS): выезд сбоку, лёгкое подпрыгивание на «верно».
